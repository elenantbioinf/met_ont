# Clair3 models

This directory contains external Clair3 models required for variant calling.

These models are not generated by the pipeline and must be downloaded separately when needed.

Current model used in this project:
- `r1041_e82_400bps_hac_v500`

Source:
- https://github.com/nanoporetech/rerio/tree/master/clair3_models

The script used to download and prepare the model is:
- `scripts/resources/download_clair3_model.sh`
